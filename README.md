# KFU – Kişisel Finans Uygulaması

Gelir ve giderlerinizi takip edin, birikim hedefleri belirleyin ve finansal alışkanlıklarınızı geliştirin.

🌐 **[egekfu.com](https://egekfu.com)** &nbsp;|&nbsp; 🚀 **[Uygulamayı Dene](https://egekfu.com/app)**

---

## Özellikler

- Gelir & gider takibi
- Kategori bazlı harcama analizi
- Birikim hedefleri
- Aylık raporlar ve grafikler
- Kart yönetimi
- Tamamen ücretsiz, veriler cihazda kalır

## Teknolojiler

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Recharts
- React Router

## Kurulum

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

GitHub Pages üzerinden [egekfu.com](https://egekfu.com) adresinde yayınlanır.
